# Phase J Cleanup Guide

## What to upload

Upload the contents of this folder directly into the repository root:

- `.nojekyll`
- `index.html`
- `assets/`
- `cases/`
- `data/`
- `methods/`
- `shared/`
- `teaching/`
- `PHASE_J_POST_DEPLOYMENT_REPORT.md`

This matches the current repository style, where GitHub Pages is publishing from `/root`.

## What not to upload publicly unless you want to

- ZIP files
- old temporary standalone HTML test files
- old package guides
- local preview files

## Current GitHub Pages setting

Use:

- Branch: `main`
- Folder: `/root`

- `/cases/higher-education-ips.html`
- `/cases/2008-financial-crisis.html`
- `/cases/geo-florida-prison.html`
- `/cases/florida-nursing-homes-visible-parasite.html`

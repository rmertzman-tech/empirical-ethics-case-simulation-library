Navigator Boundary

The EECSL can include Navigator bridge prompts, but it should not become the Navigator. Students should not enter private personal details into public case pages.

Standard bridge language:

Use this public case structure privately. Do not enter private personal details into this public case page. Use the Navigator, private notes, or your course notebook for personal application.
